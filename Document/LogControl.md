#LogControl介绍

##介绍
	本控制器中包含用户登录、注册、密码找回等安全性功能
	
##接口

###接口总览
|        接口         |  方法  |  数据  |  返回  |  作用  |
| :---------------: | :--: | :--: | :--: | :--: |
|  /api/Log/login   | post | json | json | 用户登录 |
| /api/Log/register | post | json | json | 用户注册 |
|  /api/Log/forget  | get  |  url   | json | 密码找回 |

###接口详情

####/api/Log/login

方法：POST

数据样例

```
data:{
	"userid":"17854290633",
	"password":"123456",
	"logtype":0 //登录方式：0代表手机，1代表邮箱
}
```
返回样例:`{ "result" : 代码}`

|  代码  |    含义    |
| :--: | :------: |
|  0   | 失败，用户不存在 |
|  1   |    成功    |
|  2   | 失败，密码错误  |

####/api/Log/register

方法：POST

数据样例

```
data:{
	"userid":"17854290633",
	"password":"123456",
	"username":"请叫我欧尼酱",
	"code":"000688", //短信验证
	"registertype":0 //登录方式：0代表手机，1代表邮箱
}
```
返回样例： `{ "result" : 代码}`

|  代码  |       含义        |
| :--: | :-------------: |
|  0   |    失败，验证码错误     |
|  1   |       成功        |
|  2   | 失败，该账户已存在（学号存在） |
